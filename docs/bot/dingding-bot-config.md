# DingTalk Enterprise Bot Configuration

## DingTalk Bot
DingTalk bot message receiving requires the Enterprise Bot capability
https://open.dingtalk.com/document/dingstart/configure-the-robot-application

Message receiving supports two modes: `HTTP mode` (requires public URL configuration) and `Stream mode`. Stream mode is recommended.

Application creation steps: https://open.dingtalk.com/document/dingstart/create-application

Application Development > Enterprise Internal Application > DingTalk Application > Create Application > Add Application Capability > Bot
